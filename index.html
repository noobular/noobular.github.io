Further Explanation on Copyright and Creative Commons

Importance of Copyright, Creative Commons, and Fairuse.

It's very important for a creator, being a youtuber, developer, maker because it can be the reason the product they create is okay, or illegal.
Creative commons is more related for youtubers or people making educational videos because it allows you to grab informational things or parody things without being sued because small bits of intellectual property was used.

It's also important for Game Developers to follow the guidelines so their games can be monetized along with to not get sued.

Description of Fairuse

As defined by copyright.gov, Fair use is a legal doctrine that promotes freedom of expression by permitting the unlicensed use of copyright-protected works in certain circumstances.
Section 107 of the Copyright Act provides the statutory framework for determining whether something is a fair use.
Section 107 calls for consideration of the following four factors in evaluating a question of fair use:
      Purpose and character of the use, including whether the use is of a commercial nature or is for nonprofit educational purposes
      Nature of the copyrighted work
      Amount and substantiality of the portion used in relation to the copyrighted work as a whole
      Effect of the use upon the potential market for or value of the copyrighted work
Source: Wikipedia


Description of Public Domain

Public Domain is the state of belonging or being available to the public as a whole, and therefore not subject to copyright.
There are four common ways that works arrive in the public domain:
      The copyright has expired
      The copyright owner failed to follow copyright renewal rules
      The copyright owner deliberately places it in the public domain, known as “dedication”.
      Copyright law does not protect this type of work.
Examples of items that are in the public domain include: Titles and processes,works by the Federal Government, works that were never covered by the property laws, and works where copyright has expired.


Description of Creative Commons

Creative Commons is a Non-Profit organization, that it's aim is to increase the amount of content people have to LEGALLY expand on, and also share.
Creative Commons along with Fairuse is highly talked about in the youtube community because of false copyright strikes that have been starting to swarm the website.

Git and why it's useful

What is Git?

Git is a version control system for tracking changes in computer files and coordinating work on those files among multiple people. It is primarily used for source code management in software development, but it can be used to keep track of changes in any set of files. As a distributed revision control system it is aimed at speed, data integrity, and support for distributed, non-linear workflows.

Who created Git and why?

Git was created by Linus Torvalds in 2005 for development of the Linux kernel, with other kernel developers contributing to its initial development. Its current maintainer since 2005 is Junio Hamano.
As with most other distributed version control systems, and unlike most client–server systems, every Git directory on every computer is a full-fledged repository with complete history and full version tracking abilities, independent of network access or a central server.
Git is free and open source software distributed under the terms of the GNU General Public License version 2.

Github and why it's useful

What is Github?

GitHub (originally known as Logical Awesome LLC)is a web-based hosting service for version control using git. It is mostly used for computer code. It offers all of the distributed version control and source code management (SCM) functionality of Git as well as adding its own features. It provides access control and several collaboration features such as bug tracking, feature requests, task management, and wikis for every project.

What was the goal?

The goal was to create a planter that would have an integrated cheap system for automatically watering your plants to give you more time to do other activities.

Major Milestones

The most important milestones would be getting the planter physically built, so woodworking, and drilling out the PVC pipe for the cup holes.
The actual planning that went into this was from trello, and then picking out who was doing what work.

Documentation included pictures, and explaining what was happening on trello and discord.
The major goals of the project were to finish the planter itself, without electronics.
Finish the electronics where they would work if given the correct sensors and other electronic doodads.
Finally the fact that everything was kept documented on our work progress similar to an agile scheme.

Timeline

First we had to come up with an idea (Ryan, Tyler, and I).
Then we needed to come up with the materials, I had some laying around in my garage and then got a tube from the WDF Construction team.
After that we theory crafted in how we would actually do this in terms of code, so would we do this based on sensors or just based on time?

We used multiple things to get in contact and organize the project, a big organizing tool we used was Trello, and then Discord for communication.

Making the Pi Functional

The process of making the Pi usable is pretty simple, First you start out with at least an 8gb micro sd card, you plug that into an sd card reader either USB or a hub connected to a computer.
You can then download the most updated version of Rasbian OS from the Raspberry Pi website.
Once you've got that you can download Etcher, a disk imaging software, which will essentially write the os onto the sd card allowing you to boot from it.
once that's finished all you need to do is plug it into the pi and give the pi some power, and you're ready to go.
